# Split-Brain IoT Glove Wiring Guide

This document specifies the exact pin connections for the Arduino Nano and ESP8266 pair that make up the SYNAPSE Smart Glove telemetry hub.

## 1. Core Architecture (Split-Brain)
- **Sensor Reader:** Arduino Nano (5V logic, handles 6 analog inputs + I2C)
- **WiFi Gateway:** ESP8266 NodeMCU (3.3V logic, handles WebSocket streaming)
- **Bridge:** Serial UART (Nano TX -> Voltage Divider -> ESP8266 RX)

## 2. Power Distribution
| Source | Destination | Pin | Notes |
|--------|-------------|-----|-------|
| 5V USB (or Battery) | Arduino Nano | `VIN` | Primary Power IN |
| Arduino Nano | ESP8266 | `5V` -> `VIN` | Powers ESP from Nano Regulator |
| Arduino Nano | Sensors | `5V` (or `3V3`) | VCC rail for all sensors |
| Common Ground | All Components | `GND` | **CRITICAL: ALL GND PINS MUST BE CONNECTED** |

## 3. Sensors -> Arduino Nano
*The Nano is required because standard Uno R3 uses A4/A5 for I2C, leaving only 4 analog pins available. The Nano exposes A6 and A7, providing enough analog pins for 5 Flex + 1 Pulse.*

### Flex Sensors (5x)
*Requires 5x 10kΩ Pull-Down Resistors (Voltage Divider configuration)*
| Sensor | Nano Pin | Notes |
|--------|----------|-------|
| Thumb | `A0` | Flex Sensor 1 |
| Index | `A1` | Flex Sensor 2 |
| Middle | `A2` | Flex Sensor 3 |
| Ring | `A3` | Flex Sensor 4 |
| Pinky | `A6` | Flex Sensor 5 |

### Pulse Sensor
| Sensor Pin | Nano Pin | Notes |
|------------|----------|-------|
| Signal | `A7` | Heart rate analog in |
| VCC | `5V` | |
| GND | `GND` | |

### MPU6050 (Gyro/Accel)
| MPU6050 Pin | Nano Pin | Notes |
|-------------|----------|-------|
| VCC | `5V` | |
| GND | `GND` | |
| SCL | `A5` | I2C Clock |
| SDA | `A4` | I2C Data |

## 4. Arduino Nano <-> ESP8266 (Serial Bridge)
*Because the Nano operates at 5V and the ESP8266 RX pin is 3.3V, a voltage divider is required to prevent damaging the ESP.*

### Voltage Divider (Nano TX -> ESP RX)
Use a 1kΩ and 2kΩ resistor to step down 5V to ~3.3V.
```
Nano TX (D1) ----[ 1kΩ ]----+---- ESP8266 RX (D9 / RX0)
                            |
                         [ 2kΩ ]
                            |
                           GND
```

### Connection Summary
| Nano Pin | Intermediary | ESP8266 Pin | Notes |
|----------|--------------|-------------|-------|
| `TX` (D1)| Voltage Divider| `RX` (RX0) | Main data flow (Sensors -> WiFi) |
| `RX` (D0)| Direct Line | `TX` (TX0) | (Optional) Feedback / Control |
| `GND` | Direct Line | `GND` | Must be common |

*Note for developers: The CSV format generated by the Nano over TX will be: `<F1,F2,F3,F4,F5,GX,GY,HR>\n`*
